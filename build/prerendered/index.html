This an example project to show the function from the erech24 API v2 in sveltekit without sync/push functionality. This project get the data from the erech24 API v2 and displays it.

The user or an refresh from the page will not trigger directly an request to the API server. Only after an treshold time the data will be updated and compared with the local saved version. In case a new version exists on erecht24 it'll be saved on the server.

The project is not optimized for production just for demonstration how it works.
